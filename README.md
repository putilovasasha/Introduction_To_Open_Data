# Анализ данных NASA об астероидах и кометах 

## О проекте 
Проект посвящен анализу данных о приближениях астероидов и комет к Земле. 

**Цель проекта:** изучить динамику сближений астероидов и комет относительно Земли, используя официальный *API NASA*, а конкретно — *SBDB Close-Approach Data API.* 

**Задачи проекта:**
1. Сбор и визуализация данных о приближениях комет и астероидов к Земле в период с 3.12.2025 по 3.01.2026
2. Сравнение самых близких объектов к Земле по физическим параметрам с помощью запроса к *SBDB (Small-Body DataBase)*
3. Сбор и визуализация данных по самым близким к Земле объектам за весь 2025 год

## Что сделано
1. Получение и фильтрация данных через API нескольких датасетов (SBDB Close-Approach Data API, Small-Body DataBase)
2. Предобработка данных для выполнения поставленных задач: очистка, объединение датафреймов, а также конвертация в привычные единицы для удобства в вычислениях
3. Визуализация данных с помощью matplotlib

## Используемые технологии
* Jupyter Notebook
* Библиотеки: pandas, numpy, matplotlib, requests, time
